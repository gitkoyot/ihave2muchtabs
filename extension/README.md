# I Have 2 Much Tabs Extension (MVP Skeleton)

Plugin-only MVP skeleton (no local companion service).

## Included

- Manifest V3 setup
- Background service worker bootstrap
- Popup / Options / Dashboard entry points
- TypeScript modules for storage, bookmarks, prompts, and Azure OpenAI integration

## Planned Build

```bash
npm install
npm run build
```

Load `extension/` as an unpacked extension in Chrome.
